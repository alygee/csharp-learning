# csharp-learning
##### День 1–10: База C#
###### День 1
- Установить .NET SDK, IDE (Rider/VS Code/Visual Studio).
- Команды: dotnet --info, dotnet new console, dotnet run.
- Практика: “Hello, C#”, вывод переменных.
###### День 2
- Типы данных, var, nullable, преобразования типов.
- Практика: программа-конвертер (валюты/температуры).
###### День 3
- Условия: if, switch, тернарный оператор.
- Практика: калькулятор с меню операций.
###### День 4
- циклы (`for`, `while`, `foreach`)
###### День 5
- Методы, параметры, ref/out, перегрузка.
- Практика: вынести калькулятор в методы.
###### День 6
- Массивы, List<T>, Dictionary<TKey, TValue>.
- Практика: учет книг (добавить/удалить/найти).
###### День 7
- Строки, StringBuilder, работа с датой и временем.
- Практика: парсер строки заказа.
###### День 8
- ООП: классы, свойства, конструкторы, инкапсуляция.
- Практика: User, Product, Order.
###### День 9
- Наследование, интерфейсы, полиморфизм.
- Практика: система оплат (ICheckout, CardPayment, CashPayment).
###### День 10
- Обработка исключений, try/catch/finally.
- Практика: безопасный ввод и логирование ошибок в файл.
